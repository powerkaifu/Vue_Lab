模板語法

{{ person }}：{{ msg }}

{{}} 裡面可以使用表達式 expression

  • {{ num + 100 }}
  • {{ true ? '天氣好' : '天氣糟' }}
  • {{ msg || '哈囉' }}
  • {{ msg.split('').reverse().join('') }}

語句（Statement）、流程控制（flow control）都無法在鬍子語法裡面

{{ once }}

綁屬性 v-bind，縮寫「:」

  • 如果綁定的值是 null 或 undefined，那麼該 attribute 將不會被包含在渲染的元素上。
  • 值被判斷 truty 或空字串都可以被渲染。
