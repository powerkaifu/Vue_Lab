表單輸入綁定

  • v-model 指令可在表單 input、textarea 及 select 元素上創建雙向資料綁定，它會根據控件類型自動選取正確的方法來更新元素。
  • v-model 會忽略所有表單元素的 value、checked、selected attribute 的初始值，將當前活動實體的資料作為來源。
  • 通過 JavaScript 在組件的 data 選項中宣告初始值。
雙向綁定：指資料可以流出到畫面，同時，也可以寫入到實體設定的資料屬性。

v-model 在內部為不同的輸入元素使用不同的 property 並拋出不同的事件：

  • text 和 textarea 元素使用 value property 和 input 事件。
  • checkbox 和 radio 使用 checked property 和 change 事件
  • select 字段將 value 作為 prop 並將 change 作為事件。
